# Create Codex Rules

Generate or update `AGENTS.md` from codebase analysis.

## Read

- `package.json`
- `tsconfig.json`
- `README.md`
- docs in `docs/`
- key source folders

## Capture

- what the project is
- technologies used
- structure
- commands
- conventions
- security, settings, and voice constraints

Keep the result compact and repo-specific.
